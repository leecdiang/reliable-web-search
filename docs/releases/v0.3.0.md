## English

### reliable-web-search v0.3.0 — Unified CLI, Interactive Setup, MCP Server

This release introduces the `rws` CLI, interactive setup wizard, stdio-based MCP server, and agent host adapters for OpenClaw, Codex, and Claude Code. Search is now configurable at the command line in one continuous flow.

#### Highlights

- **Unified CLI (`rws`)** — single entry point for setup, search, agent registration, and diagnostics
- **Interactive setup wizard** — provider selection, hidden API key input, connection verification, agent detection, multi-host installation
- **Stdio MCP server (`rws mcp`)** — exposes `reliable_web_search` via `@modelcontextprotocol/sdk`
- **Agent adapters** — OpenClaw (`verified`), Codex (`beta`), Claude Code (`beta`), Generic MCP (`standard`)
- **Proxy support** — `HTTP_PROXY`, `HTTPS_PROXY`, `NO_PROXY` via `undici` `EnvHttpProxyAgent`
- **Local config storage** — `config.json` + `credentials.json` (0600 on Unix), atomic writes
- **`rws doctor`** — Node.js, config, credentials, providers, MCP, and agent health checks
- **8 packaged CLI/MCP smoke tests**

#### Install

```bash
npm install --global reliable-web-search
```

#### Quick Start

```bash
rws setup
rws "latest RISC-V news"
rws doctor
```

## 简体中文

### reliable-web-search v0.3.0 — 统一 CLI、交互式配置、MCP 服务器

本版本引入了 `rws` CLI、交互式配置向导、基于 stdio 的 MCP 服务器，以及 OpenClaw、Codex 和 Claude Code 的 Agent 适配器。现在可以在一个连续流程中通过命令行完成搜索配置。

#### 亮点

- **统一 CLI（`rws`）** ——配置、搜索、Agent 注册、诊断的单一入口
- **交互式配置向导**——供应商选择、隐藏的 API Key 输入、连接验证、Agent 检测、多宿主安装
- **Stdio MCP 服务器（`rws mcp`）**——通过 `@modelcontextprotocol/sdk` 暴露 `reliable_web_search`
- **Agent 适配器**——OpenClaw（已验证）、Codex（Beta）、Claude Code（Beta）、Generic MCP（标准）
- **代理支持**——通过 `undici` `EnvHttpProxyAgent` 支持 `HTTP_PROXY`、`HTTPS_PROXY`、`NO_PROXY`
- **本地配置存储**——`config.json` + `credentials.json`（Unix 0600），原子写入
- **`rws doctor`**——Node.js、配置、凭据、供应商、MCP 和 Agent 健康检查
- **8 个打包 CLI/MCP 冒烟测试**

#### 安装

```bash
npm install --global reliable-web-search
```

#### 快速开始

```bash
rws setup
rws "latest RISC-V news"
rws doctor
```
